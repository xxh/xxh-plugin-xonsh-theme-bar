# xxh-plugin-theme-bar
Bar theme plugin for [xonssh/xxh](https://github.com/xxh/xxh). Install:

```
cd ~/.xxh/xxh/plugins
git clone --depth 1 https://github.com/xxh/xxh-plugin-xonsh-theme-bar
# connect once with updating host
xxh yourhost +if
```
